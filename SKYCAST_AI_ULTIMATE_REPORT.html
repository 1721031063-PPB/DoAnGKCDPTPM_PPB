1Tổng Quan Sản Phẩm

SKYCAST AI là ứng dụng web thời tiết thế hệ mới, kết hợp dữ liệu thời tiết thực từ API toàn cầu với sức mạnh của trí tuệ nhân tạo (GPT-4o-mini) để cung cấp trải nghiệm thời tiết cá nhân hóa, thông minh và thân thiện với người Việt.

Khác biệt với các ứng dụng thời tiết truyền thống chỉ hiển thị số liệu khô khan, SKYCAST AI biến dữ liệu thành trợ lý thời tiết cá nhân — tóm tắt bằng ngôn ngữ tự nhiên, gợi ý trang phục, phân tích giờ vàng xuất phát và cảnh báo sức khỏe toàn diện.

6Chi Tiết API Routes

POST /api/weather

Mục đích: Lấy toàn bộ dữ liệu thời tiết hiện tại, dự báo 5 ngày và nhiệt độ 24 giờ.

Thuộc tínhChi tiết
Input{ city: string }
Output{ current, forecast[], hourly[], lat, lon }
Đặc biệtTự tính UV Index từ % mây + giờ trong ngày; Dịch tự động 50+ mô tả Anh → Việt
Ví dụ dịch: "few clouds" → "Ít mây" · "thunderstorm" → "Dông" · "light rain" → "Mưa nhẹ" · "overcast clouds" → "Trời u ám"

POST /api/weather-summary

Mục đích: Tóm tắt thời tiết bằng ngôn ngữ tự nhiên tiếng Việt (~80 từ).

Thuộc tínhChi tiết
Input{ city, current, forecast[], health }
Luồng xử lýGPT-4o-mini (nếu có key) → Fallback generateFallbackSummary() tự sinh câu
Đảm bảoApp không bao giờ crash dù OpenAI lỗi hoặc hết quota

POST /api/ai-planner

Mục đích: AI Cố Vấn Du Lịch — phân tích kế hoạch người dùng dựa trên thời tiết thực.

Thuộc tínhChi tiết
Input{ activity, timeDescription, current, forecast }
Output (JSON Mode){ score: 85, outfit: [...], bestTime: "6–8h", advice: "..." }
FallbackThuật toán tính điểm từ nhiệt độ (gần 25°C = tối ưu) + xác suất mưa

GET / POST / DELETE /api/favorites

Yêu cầu: NextAuth Session hợp lệ (trả 401 nếu chưa đăng nhập).

MethodHành động
GETTrả về danh sách yêu thích của user hiện tại từ MongoDB
POST { label, lat, lon }Thêm địa điểm mới vào embedded favorites array
DELETE ?label=...Xoá địa điểm khỏi mảng yêu thích của user

POST /api/history

Thuộc tínhChi tiết
NguồnOpen-Meteo — Hoàn toàn miễn phí, không cần API key
Input{ lat, lon }
OutputMảng 30 object: { date, maxTemp, minTemp, precipitation }

7Cơ Sở Dữ Liệu — MongoDB

Kết nối: mongodb://localhost:27017/skycast

💡 Tự động khởi tạo: MongoDB tự tạo database skycast và collection users khi user đầu tiên đăng ký. Không cần setup thủ công.

Mongoose Schema — User

{ name: String, // Tên hiển thị email: String, // unique index — không trùng lặp password: String, // bcrypt hash 10 rounds — KHÔNG lưu plain text favorites: [{ // Embedded array — không tách collection riêng label: String, // VD: "Hội An, Quảng Nam, Vietnam" lat: Number, // Vĩ độ lon: Number, // Kinh độ }], createdAt: Date, // timestamps: true (tự động bởi Mongoose) updatedAt: Date, }
Tại sao dùng Embedded Array? Favorites được nhúng trực tiếp vào User document thay vì tách thành collection riêng. Lý do: Mỗi user chỉ có tối đa ~20 địa điểm yêu thích, nên embedded giúp truy vấn nhanh hơn (1 query thay vì JOIN/populate), đơn giản hóa CRUD và đảm bảo data integrity.

Cấu hình môi trường (.env.local)

# ── BẮT BUỘC ───────────────────────────────────────────── NEXT_PUBLIC_OPENWEATHER_API_KEY=... # openweathermap.org MONGODB_URI=mongodb://localhost:27017/skycast NEXTAUTH_SECRET=chuoi_bi_mat_bat_ky NEXTAUTH_URL=http://localhost:3000 # ── TUỲ CHỌN (có Fallback khi thiếu) ───────────────────── OPENAI_API_KEY=... # GPT-4o-mini NEXT_PUBLIC_WAQI_TOKEN=... # AQI NEXT_PUBLIC_AMBEE_API_KEY=... # Phấn hoa
⚠️ Quan trọng: OPENAI_API_KEY là tuỳ chọn. Khi thiếu hoặc hết quota, cả weather-summary lẫn ai-planner đều tự động chuyển sang thuật toán Fallback nội bộ — app không bao giờ bị crash.
