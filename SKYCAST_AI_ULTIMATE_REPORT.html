01 Giới Thiệu & Mô Tả Sản Phẩm

04 Kiến Trúc Hệ Thống

Dự án sử dụng kiến trúc Monolithic hiện đại trên nền tảng Next.js — kết hợp Frontend và Backend trong một codebase thống nhất, tận dụng App Router để tối ưu SSR và API Routes làm Backend layer.

[Người dùng / Browser]
        ↓
[🖥️ NEXT.JS FRONTEND — CLIENT SIDE]
   ├─ src/app/components/    (Charts, Maps, Dashboard UI)
   ├─ src/app/favorites/     (Trang quản lý yêu thích /favorites)
   ├─ src/app/compare/       (Trang so sánh thành phố /compare)
   ├─ src/app/widget/        (Trang tạo mã iframe /widget)
   ├─ src/app/auth/          (Trang đăng nhập / đăng ký)
   └─ public/sw.js           (Service Worker — Push Notification)
        ↓
[🔐 NEXTAUTH MIDDLEWARE — SESSION & JWT]
        ↓
[⚙️ BACKEND API LAYER — SERVER SIDE (11 Routes)]
   ├─ /api/weather           → OpenWeatherMap + Việt hóa 50+ mô tả
   ├─ /api/weather-summary   → GPT-4o-mini tóm tắt (+ Fallback nội bộ)
   ├─ /api/ai-planner        → GPT JSON Mode (+ Fallback điểm số)
   ├─ /api/history           → Open-Meteo (30 ngày — miễn phí)
   ├─ /api/favorites         → CRUD MongoDB (bảo mật NextAuth)
   └─ /api/auth/[...nextauth]→ Đăng ký / Đăng nhập + bcrypt
        ↓
[🗄️ DATABASE — MONGODB ATLAS]
   └─ Collection: users { name, email, password, favorites[] }

Cơ Sở Dữ Liệu — User Schema

{
  "name": String, // Display Name
  "email": String (unique, indexed),
  "password": String, // hashed by bcrypt (salt: 12)
  "favorites": [
    { "label": String, "lat": Number, "lon": Number }
  ] // Embedded Document — không cần JOIN
}

Dự án thực hiện theo mô hình Agile 5 Sprint, mỗi sprint tập trung vào một nhóm tính năng cụ thể với kết quả đánh giá rõ ràng:


Sprint 1 — Foundation & Core UI (Tuần 1–2)
Khởi tạo Next.js 16 + TypeScript + Tailwind CSS + Turbopack. Thiết kế hệ thống Glassmorphism. Tích hợp OpenWeatherMap API, component tra cứu thời tiết & dự báo 5 ngày. Cài đặt Framer Motion cho animation khí quyển. Xây dựng bảng Việt hóa 50+ cụm từ.

Cấu hình MongoDB Atlas + kết nối Mongoose. Thiết kế User Schema với embedded favorites[]. Tích hợp NextAuth.js Credentials Provider + JWT Strategy. Triển khai mã hóa bcrypt (salt rounds: 12). Bảo vệ API Routes bằng getServerSession().

06 Chi Tiết API Routes (11 Endpoints)

POST /api/weather Fetch OpenWeatherMap + Việt hóa tự động mô tả (50+ mappings).
POST /api/weather-summary GPT-4o-mini sinh tóm tắt ~80 từ tiếng Việt. Có Fallback thuật toán nội bộ.
POST /api/ai-planner GPT JSON Mode tư vấn kế hoạch: điểm an toàn, trang phục, giờ vàng. Có Fallback.
GET /api/favorites Lấy danh sách favorites của user hiện tại (bảo mật bởi NextAuth Session).
POST /api/favorites Thêm địa điểm vào favorites, lưu vào MongoDB.
DEL /api/favorites Xóa địa điểm khỏi danh sách favorites.
POST /api/history Lấy dữ liệu 30 ngày lịch sử từ Open-Meteo API (nhiệt độ, lượng mưa).
POST /api/auth/register Đăng ký tài khoản mới, hash mật khẩu bcrypt trước khi lưu MongoDB.
ALL /api/auth/[...nextauth] Xử lý toàn bộ luồng NextAuth: session, callback, JWT, sign-in.

07 Đặc Tả Chi Tiết Tính Năng

🔍
Tra Cứu & Dự Báo
OpenWeatherMap API — Autocomplete địa danh, nhiệt độ, độ ẩm, gió, dự báo 5 ngày, AQI, UV Index, Phấn hoa. Việt hóa 50+ mô tả tự động.

AI Summary
GPT-4o-mini sinh đoạn văn ~80 từ tiếng Việt tự nhiên. Khi API lỗi → Fallback nội bộ tự sinh câu từ dữ liệu thô. 100% uptime đảm bảo.

AI Planner
Nhập kế hoạch → GPT JSON Mode trả về: điểm an toàn (0–100), gợi ý trang phục, khung giờ vàng, lời khuyên cụ thể.
⭐
Hệ Thống Yêu Thích
Đăng ký/Đăng nhập bcrypt + JWT. Lưu địa điểm vào MongoDB. Trang /favorites chuyên biệt với giao diện lưới, tra cứu nhanh thời tiết.
📈
Lịch Sử 30 Ngày
Open-Meteo API miễn phí cung cấp nhiệt độ & lượng mưa 30 ngày. Hiển thị qua Area Chart và Bar Chart (Recharts).
🌍
Bản Đồ 6 Lớp
Leaflet tích hợp 6 tile layer chuyển đổi realtime: Radar mưa, Nhiệt độ, Gió, Mây, Áp suất, Lượng mưa. Click xem vị trí bất kỳ.
🔔
Push Notification
Service Worker (sw.js) cảnh báo chủ động khi xác suất mưa > 70%. Giới hạn 1 lần/30 phút để tránh làm phiền.
⚖️
So Sánh Thành Phố
Nhập 2 thành phố → so sánh song song toàn bộ chỉ số với bảng chấm điểm tự động thành phố "dễ sống hơn".
🧩
Widget Nhúng
Chọn thành phố + theme màu sắc (Ocean, Forest, Sunset) → sinh mã <iframe> nhúng vào bất kỳ website nào.

09 Giới Hạn & Roadmap

Giới Hạn Hiện Tại

OpenAI Quota
Free tier có giới hạn tokens/tháng. Giải pháp: Fallback nội bộ đã xử lý hoàn toàn — nâng cấp credit khi cần mở rộng.
Push Notification chỉ Client-side
Service Worker không có server trigger — cần tích hợp Web-push Protocol + Cron job định kỳ để cảnh báo proactive.
UV Index tính gần đúng
Không dùng OneCall API v3 (tốn phí). Giải pháp: Nâng cấp lên OpenWeather OneCall v3 lấy UV Index thực từ vệ tinh.

git clone https://github.com/1721031063-PPB/DoAnGiuaKy.git
cd skycast-ai

# 2. Cài đặt dependencies

npm run dev
