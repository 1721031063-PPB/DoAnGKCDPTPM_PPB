05. Kiến Trúc Hệ Thống

Dự án sử dụng kiến trúc Monolithic hiện đại trên Next.js — kết hợp Frontend và Backend trong một codebase thống nhất, tận dụng App Router để tối ưu SSR và API Routes làm Backend layer.

[Người dùng / Browser]
        ↓
[🖥️ NEXT.JS FRONTEND — CLIENT SIDE]
   ├─ src/app/components/    (Charts, Maps, Dashboard UI)
   ├─ src/app/favorites/     (Trang quản lý yêu thích)
   ├─ src/app/compare/       (Trang so sánh thành phố)
   ├─ src/app/widget/        (Trang tạo mã iframe nhúng)
   ├─ src/app/auth/          (Trang đăng nhập / đăng ký)
   └─ public/sw.js           (Service Worker — Push Notification)
        ↓
[🔐 NEXTAUTH MIDDLEWARE — SESSION & JWT]
        ↓
[⚙️ BACKEND API LAYER — SERVER SIDE]
   ├─ /api/weather           → OpenWeatherMap + Việt hóa 50+ mô tả
   ├─ /api/weather-summary   → GPT-4o-mini + Fallback nội bộ
   ├─ /api/ai-planner        → GPT JSON Mode + Fallback điểm số
   ├─ /api/history           → Open-Meteo (30 ngày miễn phí)
   ├─ /api/favorites         → CRUD MongoDB (bảo mật NextAuth)
   └─ /api/auth/[...nextauth]→ Đăng ký / Đăng nhập + bcrypt
        ↓
[🗄️ DATABASE — MONGODB ATLAS]
   └─ Collection: users { embedded favorites[] }

06. Cách Thực Hiện — Quy Trình Phát Triển

Dự án thực hiện theo mô hình Agile 5 Sprint, mỗi sprint tập trung vào một nhóm tính năng cụ thể:


Sprint 1 — Foundation & Core UI (Tuần 1–2)
Khởi tạo Next.js 16 + TypeScript + Tailwind. Thiết kế Glassmorphism UI. Tích hợp OpenWeatherMap API, component tra cứu thời tiết, dự báo 5 ngày. Cài đặt Framer Motion cho animation khí quyển (mưa, tuyết, mây). Xây dựng bảng Việt hóa 50+ cụm từ.
Sprint 2 — Authentication & Database (Tuần 3)
Cấu hình MongoDB Atlas + kết nối Mongoose. Thiết kế User Schema với embedded favorites[]. Tích hợp NextAuth.js Credentials Provider + JWT. Triển khai mã hóa bcrypt. Bảo vệ API Routes bằng getServerSession().
Sprint 3 — Data & Localization (Tuần 4)
Tích hợp Open-Meteo API lấy lịch sử 30 ngày miễn phí. Xây dựng biểu đồ Recharts (Area Chart 24h, Bar Chart 30 ngày). Tích hợp Leaflet + 6 tile layer thời tiết. Hoàn thiện 100% Việt hóa giao diện và dữ liệu API.
Sprint 4 — AI Integration & Reliability (Tuần 5)
Tích hợp OpenAI SDK GPT-4o-mini để sinh tóm tắt thời tiết tự nhiên. Xây dựng AI Planner với JSON Mode. Phát triển hệ thống Fallback nội bộ (aiSummary.ts) — tự sinh nội dung khi OpenAI không khả dụng. Kiểm tra toàn diện các edge case.
Sprint 5 — Ecosystem & Advanced (Tuần 6)
Xây dựng Trang Favorites (/favorites), Compare (/compare), Widget (/widget). Triển khai Web Push Notification qua Service Worker. Responsive Design Mobile → 4K. Đẩy code GitHub, quay video demo, hoàn thiện tài liệu.
